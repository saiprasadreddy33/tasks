# Effortless Tasks

A modern, minimalist task management application built with React, TypeScript, and Tailwind CSS.

## Features

-  Create, edit, and delete tasks
-  Priority levels (Low, Medium, High)
-  Due date tracking
-  Search and filter tasks
-  Sort by date, priority, or due date
-  Beautiful glassmorphism UI
-  Fully responsive design
-  Drag-and-drop reordering
-  Local storage persistence

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
- React Beautiful DnD

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Usage

1. Add tasks using the input field at the top
2. Set priority and due dates for each task
3. Filter tasks by status (All, Active, Done)
4. Sort tasks by creation date, priority, or due date
5. Drag and drop to reorder tasks
6. Mark tasks as complete or delete them
